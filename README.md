# High Stakes

Premium Texas Hold'em vs AI — play-money tables with a casino-style UI.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run build` — production build
- `npm run verify` — engine checks (side pots, hand evaluation)
- `npm run lint` — ESLint

## Deploy

- **GitHub:** [ikan-tech1/high-stakes-poker](https://github.com/ikan-tech1/high-stakes-poker)
- **Vercel:** [poker-ikan-tech1.vercel.app](https://poker-ikan-tech1.vercel.app)
